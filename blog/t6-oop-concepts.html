Object Oriented versus Functional Programming

September 6, 2015

There are different programming languages out there. Two popular styles of programming, are functional and Object Oriented programming. The key difference is that the funcitonal programming deals with "functions", and object oriented programming deals with objects. What this really means will be explained below.

Object Oriented Programming
These types of languages deal with "objects". Objects are just... things! As real-world things have properties, like size, weight or color, so do objects. Objects also have "methods" that they can do.
Functional Programming

Functional Programming deals with "functions". Where Object Oriented languages deal in "things" and "stuff", Functional languages deal in numbers and equations. In Functional Programming your data should always stay the same, and any new bit of information. If your function works once, it will work again and again.
